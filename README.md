# 🌦️ Weather App

A modern Weather Application built using HTML, CSS, and JavaScript (DOM) that shows real-time weather data using an API.

---

## 🚀 Features

- 🔍 Search weather by city name  
- 📍 Get weather using current location  
- 🌡️ Display temperature, condition, wind speed  
- 🌤️ Weather icons (sun, rain, clouds)  
- 📅 5-day weather forecast  
- 🔄 Auto-suggestions for city search  
- ⚠️ Error handling (invalid city, network issues)  
- 🎨 Responsive UI with modern design  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript (DOM Manipulation)  
- Open-Meteo API
